# Score System (Firebase Real-time)

เว็บแอปหน้าเดียวสำหรับบันทึกและจัดอันดับคะแนนเกม 4 ทีม (ทีมละสูงสุด 15 คน) ครอบคลุม 3 เกมต่อรอบ มีการซิงก์ข้อมูลแบบเรียลไทม์ผ่าน Firebase Realtime Database และสำรองข้อมูลบน `localStorage` เมื่อออฟไลน์

## คุณสมบัติหลัก
- กรอก/แก้ไขชื่อทีมและชื่อผู้เล่นผ่านโมดัล ตั้งชื่อได้ก่อนเริ่มเก็บคะแนน
- กรอกคะแนนรายบุคคลต่อเกม; ระบบคำนวณคะแนนทีมและรวม 3 เกมให้อัตโนมัติ
- Scoreboard เรียงอันดับทีม, สรุปคะแนนรายเกม, ประวัติคะแนนแต่ละเกม
- ซิงก์เรียลไทม์หลายอุปกรณ์ด้วย Firebase; มีตัวบ่งสถานะซิงก์และ fallback ไป `localStorage` หากออฟไลน์
- ปุ่มรีเซ็ตล้างข้อมูลในหน่วยความจำ, localStorage และ Firebase (ถ้าเชื่อมต่ออยู่)

## โครงสร้างไฟล์
- `index.html` — หน้า UI หลักและโหลดสคริปต์
- `style.css` — สไตล์ของหน้า
- `script.js` — ลอจิกจัดการคะแนน, UI interaction, Firebase/localStorage sync
- `firebase-config.js` — ใส่ Firebase config และ initialize database (ต้องตั้งค่าเอง)
- `FIREBASE_SETUP.md` — คู่มือตั้งค่า Firebase Realtime Database และ rules

## วิธีใช้งาน
1. เปิด `index.html` ในเบราว์เซอร์ (สามารถเปิดหลายเบราว์เซอร์/อุปกรณ์เพื่อทดสอบซิงก์)
2. คลิก “✏️ กรอก/แก้ไขชื่อเล่น” เพื่อกรอกชื่อทีมและผู้เล่น
3. เลือกทีมที่ดูแล → เลือกเกม (1–3) → กรอกคะแนนรายคน → “บันทึกคะแนน”
4. ดูอันดับทีม, สรุปคะแนนรายเกม และประวัติในหน้าหลัก
5. หากต้องการล้างข้อมูลทั้งหมด กด “รีเซ็ตทั้งหมด”

## เครื่องมือ/เทคโนโลยี
- Frontend: HTML + CSS + Vanilla JS
- Realtime backend: Firebase Realtime Database (ผ่าน CDN SDK compat)
- Persistence ฝั่งไคลเอนต์: `localStorage` เป็นแบ็กอัป/ออฟไลน์
